# TravelPartnerFinder

A full-stack travel companion platform that helps users find compatible travel partners, create trips, manage travel requests, and connect with fellow travelers.

## Features

- Secure User Authentication (Signup/Login)
- Create, Edit, and Manage Trips
- Discover Popular Travel Destinations
- Send and Accept Travel Requests
- Personalized User Profiles
- Real-Time Notifications
- Protected Routes and Secure Access
- Responsive and Modern UI
- In-App Chat Messaging
- Voice Calling Support
- AI-Generated Travel Itineraries

## Tech Stack

### Frontend

* React.js
* CSS3

### Backend

* Node.js
* Express.js

### Database

* MySQL
